# Swag Labs Testing Project

This project is part of the DEPI graduation initiative.  
It demonstrates **manual and automation testing** for the Swag Labs website using **Java, Selenium WebDriver, TestNG, and Maven**.

## Features
- Manual testing of Swag Labs website flows  
- Automated test scripts using Selenium WebDriver  
- TestNG used for test execution and reporting  
- Maven for project management and dependencies  
- Covers login, product selection, cart, and checkout functionalities

---

## Tech Stack
- **Programming Language:** Java  
- **Automation Tool:** Selenium WebDriver  
- **Testing Framework:** TestNG  
- **Build Tool:** Maven  
- **Version Control:** Git & GitHub
